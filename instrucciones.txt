APRENDER A COMBINAR BLOC_BUILDER CON BLOC_LISTENER
===================================================

1. En el home_screen.dart, cortamos el listener y lo colocamos dentro del BlocBuilder y luego cambiamos el BlocBuilder
por BlocConsumer:

child: Center(
  child: BlocConsumer<CounterBloc, CounterState>(
    listener: (context, state) {
      if (state is IncrementState) {
        ScaffoldMessenger.of(context).showSnackBar(const SnackBar(
          content: Text('Successfully increased'),
          duration: Duration(milliseconds: 300),
        ));
      } else if (state is DecrementState) {
        ScaffoldMessenger.of(context).showSnackBar(const SnackBar(
          content: Text('Successfully decreased'),
          duration: Duration(milliseconds: 300),
        ));
      }
    },
    builder: (context, state) {
      return Text(
        'Counter value: ${state.counterValue}',
        style: const TextStyle(
            fontSize: 20, fontWeight: FontWeight.bold),
      );
    },
  ),
)),
