APRENDER A DEBUGEAR LOS ESTADOS
================================

1. En la carpeta bloc/ creamos un archivo llamado my_bloc_observer.dart e implementamos las funciones así:

  import 'package:bloc/bloc.dart';

  class MyBlocObserver extends BlocObserver {
    @override
    void onEvent(Bloc bloc, Object? event) {
      super.onEvent(bloc, event);
      print('${bloc.runtimeType}');
    }

    @override
    void onError(BlocBase bloc, Object error, StackTrace stackTrace) {
      super.onError(bloc, error, stackTrace);
      print(error);
    }

    @override
    void onChange(BlocBase bloc, Change change) {
      super.onChange(bloc, change);
      print(change);
    }

    @override
    void onTransition(Bloc bloc, Transition transition) {
      super.onTransition(bloc, transition);
      print(transition);
    }
  }

2. En el archivo main.dart, vamos a cambiar la funcion msin para poder usar el observer creado anteriormente:

  void main() {
    Bloc.observer = MyBlocObserver();
    runApp(const MyApp());
  }

3. En el archivo bloc:imports.dart, agregamos la exportación de my_bloc_observer:

  export 'package:counter_app_bloc/bloc/my_bloc_observer.dart';

4. Ya en la consola de depuración, aparece los mensajes del observer cuando se hace clic en los botones:

  I/flutter ( 5753): CounterBloc
  I/flutter ( 5753): Transition { currentState: CounterInitial(0), event: Instance of 'CounterIncrementEvent', nextState: CounterState(1) }
  I/flutter ( 5753): Change { currentState: CounterInitial(0), nextState: CounterState(1) }
